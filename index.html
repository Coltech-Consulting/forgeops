Welcome to ForgeOps (ForgeRock DevOps)

ForgeRock provides a number of resources to help you get started in the cloud. These resources demonstrate how to deploy the ForgeRock Identity Platform on Kubernetes.

Start Here

Important information you should know before deploying on Kubernetes.

forgeops Repository

How to use ForgeRock’s public Git repository containing Docker and Kubernetes artifacts.

Cloud Developer’s Kit (CDK)

A minimal deployment you use to build customized Docker images for the platform.

Cloud Deployment Model (CDM)

ForgeRock’s reference implementation for cloud deployments. The CDM is not a production deployment.

Deployment Topics

Base images, backup, monitoring, security, and benchmarking for ForgeRock cloud deployments.

Support options for the ForgeRock Identity Platform and ForgeRock’s DevOps offering.
